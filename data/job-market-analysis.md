## 三、各城市AI产业聚集情况

### 3.1 北京 🥇 — 中国AI之都

**核心区域**：海淀（中关村/五道口/西二旗）、朝阳（望京）、亦庄

| 类型 | 代表企业 |
|------|------|
| 大厂AI Lab | 百度（文心）、字节（豆包/扣子）、美团、快手、京东 |
| AI创业独角兽 | 智谱AI（GLM）、月之暗面（Kimi）、百川智能、MiniMax、零一万物、面壁智能 |
| AI Infra | 硅基流动、清程、潞晨科技 |
| Agent平台 | 扣子(Coze)、Dify（部分团队） |

**特点**：
- 人才密度全国最高（清华/北大/中科院）
- 融资环境最好，投资人扎堆
- 从基座模型→中间件→应用全链条最完整
- 竞争也最激烈

### 3.2 上海 🥈 — 金融AI + 外企 + 出海

**核心区域**：浦东（张江/陆家嘴）、徐汇（漕河泾）、杨浦（五角场）

| 类型 | 代表企业 |
|------|------|
| 大厂 | 拼多多、B站、小红书、米哈游 |
| AI创业 | MiniMax（总部）、阶跃星辰、深势科技 |
| 金融AI | 蚂蚁（部分）、各券商/银行AI部门 |
| 外企AI | 微软亚洲研究院(上海)、Tesla AI、SAP |
| Agent/应用 | Dify（部分团队）、来也科技 |

**特点**：
- 金融AI应用场景丰富
- 外企多，国际化氛围好
- 游戏/内容行业AI应用需求大

### 3.3 深圳 🥉 — 硬件+AI，端侧智能前沿

**核心区域**：南山（科技园/后海）、福田

| 类型 | 代表企业 |
|------|------|
| 大厂 | 腾讯（混元）、华为（盘古）、大疆 |
| 机器人+AI | 优必选、小鹏鹏行、逐际动力 |
| 具身智能 | 星尘智能、银河通用 |
| AI应用 | 微众银行AI、平安科技、顺丰科技 |

**特点**：
- 端侧AI/具身智能/AI硬件全国领先
- 腾讯生态丰富（元器/混元）
- 创业氛围务实，变现路径清晰

### 3.4 杭州 — 阿里生态 + 电商AI + 数据智能

**核心区域**：余杭（未来科技城）、西湖、滨江

| 类型 | 代表企业 |
|------|------|
| 大厂 | 阿里巴巴（通义千问）、蚂蚁集团、网易 |
| AI创业 | 深度求索(DeepSeek)、同花顺AI |
| 电商AI | 各种电商SaaS+AI公司 |
| 数据智能 | 个推、数澜科技 |

**特点**：
- DeepSeek崛起使杭州成为大模型重镇
- 阿里通义系列生态丰富（百炼平台）
- 生活成本低于一线，互联网文化浓厚

### 3.5 成都 — 西部AI高地，成本优势突出

**核心区域**：高新区（天府软件园）、天府新区

| 类型 | 代表企业 |
|------|------|
| 大厂分部 | 腾讯(天美)、字节、阿里、美团、京东 |
| AI创业 | 晓多科技、考拉悠然、数之联 |
| 游戏AI | 大量游戏公司AI应用岗位 |
| 政府项目 | 智慧城市/数字政务AI（大量项目） |

**政策**：成都"AI十条"政策，对AI企业给予最高1000万元资金支持。

### 3.6 南京 — AI基础设施 + 智能制造

**核心区域**：雨花台（软件谷）、江北新区、江宁

| 类型 | 代表企业 |

### 3.7 武汉 — 光谷AI + 自动驾驶

| 高校孵化 | 武汉大学/华中科大AI团队 |

### 3.8 西安 — 硬科技 + 军工AI

**核心区域**：高新区、西咸新区

| 类型 | 代表企业 |

### 3.9 苏州 — 工业AI + 机器视觉

### 7.3 2025-2026年求职关键趋势

1. **Agent方向爆发**：2025年是"AI Agent元年"，各公司大量招聘Agent开发工程师（Coze/Dify平台经验加分）
2. **多模态应用增长**：从纯文本转向图文/音视频多模态应用开发
3. **RAG仍是主流**：企业级AI应用中RAG架构最为普及
4. **远程办公增加**：部分创业公司支持远程，降低了对一线城市的依赖
5. **垂直行业AI化**：金融/医疗/法律/教育等行业AI应用岗位增长
